## Quick Start

### Clone the repo

Open up a terminal and execute the following command.

```bash
git clone https://github.com/sudiptog81/reactexpress.git
```

### Running the app

```bash
# Change the directory
cd reactexpress

# Install Dependencies
npm install

# Start a Live Development Server
npm run dev
```
